Privacy Policy

Last updated: 2 May 2026  ·  Effective date: 2 May 2026

1. Overview

This privacy policy applies to the Trumpalumps Instagram automation tool (the "App"), a personal, single-user application that publishes carousel posts to the Instagram account @trumpalumps via the Meta Instagram Graph API.

The App is operated by a private individual based in the United Kingdom and is not available to the public. It has no end users other than its developer.

2. Data We Collect

The App collects and processes only the minimum data necessary to function:

  • Instagram access token — an API credential issued by Meta that allows the App to post content to the @trumpalumps Instagram account. This token is stored locally on the developer's machine in an environment file and is never transmitted to any third party.
  • Instagram user ID — a numeric identifier for the @trumpalumps account, used solely to direct API calls to the correct account.

The App does not collect, store, or process any personal data belonging to any third party or Instagram user.

3. How We Use Data

The access token and user ID are used exclusively to authenticate API requests to Meta's Instagram platform for the purpose of publishing pre-prepared image carousel posts. No data is analysed, profiled, monetised, or shared.

4. Data Storage

All credentials are stored locally on the developer's personal computer in a plaintext environment file (`.env`). No data is stored in any cloud database, third-party service, or remote server.

5. Third-Party Services

The App uses the following third-party services solely for the purpose of publishing content:

  • Meta Instagram Graph API — to publish carousel posts. Subject to Meta's Privacy Policy.
  • Cloudinary — images are temporarily uploaded to Cloudinary to obtain a public URL required by the Instagram API, then immediately deleted after posting. Subject to Cloudinary's Privacy Policy.
  • Anthropic Claude API — used to generate Instagram post captions from publicly available factual content. No personal data is submitted. Subject to Anthropic's Privacy Policy.

6. Data Retention

No user data is retained beyond the duration of a single API session. Access tokens are refreshed periodically as required by Meta and overwrite the previous token locally.

7. Your Rights (UK GDPR)

As the App processes no personal data belonging to any third party, no data subject rights are applicable to other individuals. The developer, as the sole data subject whose credentials are stored, retains full control over those credentials at all times.

If you have any questions, you may contact the developer at: dj_dan98@hotmail.com

8. Supervisory Authority

The developer is based in the United Kingdom. The relevant supervisory authority is the Information Commissioner's Office (ICO).

9. Changes to This Policy

This policy will be updated if the App's data practices change. The "Last updated" date at the top of this page will reflect any changes.
